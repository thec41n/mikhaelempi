# Gas Ngoding: Install Next.js Pertama Lu!

Halo guys! Di log kali ini, gua mau _share_ gimana caranya nge-bangun markas alias _setup project_ Next.js dari nol. Kalo lu ngerasa ngebangun web pake React biasa udah mulai keteteran ngurusin _routing_, Next.js ini ibarat lu dapet _upgrade armor_ yang auto-GG!

Sebelum mulai ritual, pastiin di laptop lu udah ke-install **Node.js** ya. Ibarat motor, Node.js itu bensinnya. Kalo belom ada, mesinnya ga bakal nyala wkwk.

## Mantra Sihirnya

Kalo Node.js udah aman, lu tinggal buka terminal kesayangan lu, terus ketik mantra pemanggil ini:

```bash
npx create-next-app@latest my-web
```

_(Catatan: `my-web` itu nama folder project lu, bebas lu mau ganti nama apa aja, asal jangan pake spasi ya guys)._

Begitu lu _Enter_, si Next.js bakal nanya-nanya beberapa _setting_-an ala NPC ngasih _quest_ di awal _game_. Kalo bingung, pilih aja `Yes` buat semuanya, apalagi buat opsi **TypeScript**, **Tailwind CSS**, sama **App Router**. Tiga itu udah _combo_ maut _developer_ jaman _now_ guys!

## Nyalain Mesin!

Kalo _loading download_-nya udah kelar (ini tergantung amal ibadah dan _speed_ Wi-Fi lu yak), lu tinggal masuk ke foldernya dan nyalain _server_ lokalnya:

```bash
cd my-web
npm run dev
```

Sekarang lu buka browser dan ketik `http://localhost:3000`. Kalo muncul logo Next.js yang muter-muter, selamat guys! Lu udah resmi naik level! 🎉
